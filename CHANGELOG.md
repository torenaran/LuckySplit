# Changelog

## 1.0.0

- First public release.
- Added reusable venue presets and guided first-launch setup.
- Added nearby-player selection with name and home-world autofill.
- Added automatic ticket ranges, player limits, live pot totals, projected prize, and venue share.
- Added searchable purchase ledger and drawing archive.
- Added confirmation-protected close, draw, void, and archive actions.
- Added reproducible drawing receipts and verification details.
- Added copyable venue announcements and CSV ledger exports.
- Added branded navigation, summary cards, welcome splash, and plugin icon.
- Improved preset management with a clearly marked active preset, one-click activation, synchronized editor details, responsive panels, visible field labels, and unclipped actions.
