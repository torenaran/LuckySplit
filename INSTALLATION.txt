LUCKY SPLIT 1.0.0 — MANUAL INSTALLATION

1. Extract the Lucky Split release ZIP to a permanent folder.
2. In FFXIV, open Dalamud Settings.
3. Select Experimental.
4. Add the extracted folder under Dev Plugin Locations.
5. Open the Dalamud Plugin Installer.
6. Enable Lucky Split under developer plugins.
7. Use /luckysplit, /lucky, or /5050.

Lucky Split does not move gil or complete trades. Venue staff must confirm payments and distribute winnings manually.

Manual installations do not update automatically. Replace the installed files when a newer release is published.
