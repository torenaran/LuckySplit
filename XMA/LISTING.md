# Lucky Split

## One-line summary

A polished host console for running organized 50/50 drawings at FFXIV venues.

## Description

Lucky Split helps venue staff run clean, organized 50/50 drawings without spreadsheets or hand-assigned ticket numbers.

Create presets for every venue or recurring event, choose nearby players from a searchable picker, record confirmed sales, and let Lucky Split assign consecutive ticket ranges automatically. The dashboard tracks the total pot, projected winner prize, venue share, tickets sold, and unique entrants in real time.

When sales end, the host can review the final ledger, close the drawing with a confirmation prompt, select a reproducible winning ticket, copy a ready-to-post winner announcement, and archive the complete record. Purchase ledgers can also be exported to CSV.

Lucky Split does not move gil, complete trades, inspect balances, or send messages automatically. Venue staff remain responsible for confirming payments and distributing winnings.

## Main features

- Reusable venue presets
- Nearby-player purchase picker
- Automatic consecutive ticket ranges
- Live pot and payout calculations
- Per-player ticket limits
- Searchable ledger and history
- Confirmation-protected host actions
- Reproducible drawing receipts
- Copyable announcements
- CSV exports
- Branded host interface and welcome splash

## Commands

`/luckysplit`, `/lucky`, `/5050`

## Installation

Download and extract the release ZIP. Add the extracted folder in **Dalamud Settings → Experimental → Dev Plugin Locations**, then enable Lucky Split under developer plugins.
