# Lucky Split 1.0.0

Lucky Split is ready for its first public release.

This release includes venue presets, nearby-player selection, automatic ticket assignment, live pot and payout tracking, confirmation-protected host controls, drawing history, reproducible receipts, CSV exports, and a fully branded interface. Preset management clearly identifies the active venue, switches presets with one click, and adapts to smaller window sizes without hiding fields or actions.

## Safety boundaries

Lucky Split records information entered by venue staff. It does not move gil, inspect balances, complete trades, or send chat automatically. Hosts are responsible for confirming every purchase and payout in game.

## Commands

`/luckysplit` · `/lucky` · `/5050`
